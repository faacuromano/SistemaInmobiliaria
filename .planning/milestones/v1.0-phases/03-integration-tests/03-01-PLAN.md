`describe('ACT-02: cancelSale — lot status reverts to DISPONIBLE')`:
- `it('cancels active sale and reverts lot to DISPONIBLE')`: Mock `saleModel.findById` (NOT prismaMock.sale.findUnique — cancelSale uses the model layer) to return `{ id: 'sale-1', lotId: 'lot-1', status: 'ACTIVA' }` (include minimal shape). Assert `prismaMock.lot.update` called with `{ where: { id: 'lot-1' }, data: { status: 'DISPONIBLE' } }`.
- `it('rejects cancellation of non-ACTIVA sale')`: Mock `saleModel.findById` to return `{ id: 'sale-1', status: 'COMPLETADA', lotId: 'lot-1' }`. Assert result `{ success: false, error: expect.stringContaining('Solo se pueden cancelar ventas activas') }`.
- `it('returns error when sale not found')`: Mock `saleModel.findById` to return `null`. Assert `{ success: false, error: 'Venta no encontrada' }`.
