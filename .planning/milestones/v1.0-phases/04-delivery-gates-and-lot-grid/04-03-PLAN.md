**Part C — Add @media print block in `globals.css`:**

Add a `@media print` section at the end of the file to handle remaining edge cases that Tailwind classes don't cover well:

```css
@media print {
  /* Force Radix Collapsible closed sections to be visible */
  [data-state="closed"] > [data-radix-collapsible-content] {
    display: block !important;
    height: auto !important;
    overflow: visible !important;
  }

  /* Clean up lot cards for print */
  body {
    -webkit-print-color-adjust: exact;
    print-color-adjust: exact;
  }
}
```

The `print-color-adjust: exact` ensures the colored left borders render on paper (browsers often strip background colors during print by default; this preserves them).

Note: The Tailwind `print:` classes handle most of the work (hiding controls, forcing blocks open). The globals.css `@media print` is a fallback for Radix's `data-state` attribute which isn't directly targetable via Tailwind utility classes.
