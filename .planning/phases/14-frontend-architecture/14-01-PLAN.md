### 5.3 Patron de Tabla con Filtros
- Pattern: Page renders Filters + Table as sibling components

- Filter state: typically URL search params or local state
- Table features: sortable columns, action dropdown per row, pagination
- Usage: sales-table + sales-filters, developments-table + developments-filters, etc.

- Structure: useForm({resolver: zodResolver(schema)}) -> Form -> FormField -> FormItem -> FormControl

- Data fetching: direct server action calls (no API routes for data)
